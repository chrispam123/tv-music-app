# TV Music App - Frontend

Aplicación web para Smart TV Samsung que muestra reloj, fecha y reproduce música aleatoria.

## Archivos

- `index.html` - Estructura principal
- `css/styles.css` - Estilos optimizados para TV
- `js/app.js` - Lógica de reloj y reproducción

## Uso Local

Abre `index.html` directamente en el navegador del TV Samsung.

## Configuración

Si cambias el endpoint del API, edita la constante `API_ENDPOINT` en `js/app.js`.

## Funcionalidades

- Reloj actualizado cada segundo
- Fecha formateada en español
- Reproducción automática de música aleatoria
- Carga siguiente canción al terminar
- Reintentos automáticos en caso de error
